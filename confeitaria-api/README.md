### Dependências:

- Node.JS 20
- Extensões 
  - Prisma
  - Docker
  - DotENV 
  - REST Client

  - ESLint

### Instruções:

```

  npm install  #apenas na primeira vez

  npm install @prisma/client  #apenas na primeira vez

  npx prisma migrate generate #apenas na primeira vez

  npx prisma migrate dev #apenas na primeira vez

  npm run start:dev

```

### Subir o banco de dados:

```
  docker compose up

```
